# CampusCare - Your Concern, Our commitment

CampusCare is a complaint management web-based platform designed to digitize, simplify, and track the complaint submission process for students and college administration.

## Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript, Tailwind CSS
* **Database:** MongoDB

## Key Features
* **User Authentication:** Secure login and registration for students, staffs and admin.
* **CRUD Operations:** Students can create, view, and track complaints.
* **Dynamic Tracking:** Real-time tracking of complaint status (e.g., Pending, In Progress, Resolved).
* **Responsive UI:** Fully optimized for desktop.
